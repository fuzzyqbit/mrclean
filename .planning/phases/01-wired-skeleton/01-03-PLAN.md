<objective>
Implement the `mrclean hook` subcommand: a one-shot stdin/stdout handler that Claude Code spawns on every hook event. Reads JSON from stdin with a 10s timeout, dispatches to per-event handlers (all no-op in Phase 1 except SessionStart and UserPromptSubmit which emit the "mrclean active" wiring banner via `additionalContext`), and exits 0 on success / 2 with structured stderr on any crash.

Purpose: This is the operator-visible wiring signal of Phase 1. After this plan, running `mrclean hook` against a synthetic UserPromptSubmit payload produces JSON output containing the "mrclean active" trace. Combined with Plan 02's installer, this is what makes the next `claude` session emit the wiring proof. Defends HOOK-05 (fail-closed exit 2 on any uncaught exception) and HOOK-06 (stdout is JSON-only — diagnostics go to stderr).

**HOOK-07 Phase 1 scope:** REQUIREMENTS.md HOOK-07 specifies a banner with `(rules: NNN, allowlist: NN)` counts. Those counts cannot exist until Phase 2 ships detection. Phase 1 delivers the **short** form (`mrclean active v{VERSION} (no-op mode — detection not yet enabled)`) that satisfies the wiring-signal intent of HOOK-07; the full format with live counts lands in Phase 2 alongside DET1-01..DET4-03 and CFG-02. This is the documented Phase 1 reduction — verify-phase MUST NOT flag this as a gap.

Output: A robust hook binary surface that satisfies HOOK-01, HOOK-05, HOOK-06, and the Phase 1 short-banner subset of HOOK-07.
</objective>

      ```typescript
      import { VERSION } from '../shared/version.js';
      import { installCrashGuards, writeFailClosedError } from './failclosed.js';
      import { readStdinWithTimeout, StdinTimeoutError } from './stdin.js';
      import { dispatch } from './dispatcher.js';
      import type { HookInput } from '../shared/types.js';

      export async function runHook(): Promise<void> {
        installCrashGuards(VERSION);
        let raw: string;
        try {
          raw = await readStdinWithTimeout(10_000);
        } catch (err) {
          if (err instanceof StdinTimeoutError) {
            // Pitfall #4 — Windows / Git Bash stdin pipe stall.
            // Exit 0 silently; never block the operator on an upstream stall.
            process.exit(0);
          }
          writeFailClosedError(err, { phase: 'stdin' });
          process.exit(2);
        }
        if (raw.trim() === '') { process.exit(0); }
        let input: HookInput;
        try {
          input = JSON.parse(raw) as HookInput;
        } catch (err) {
          writeFailClosedError(err, { phase: 'parse' });
          process.exit(2);
        }
        try {
          const result = dispatch(input);
          if (result !== null) {
            process.stdout.write(JSON.stringify(result));
          }
          process.exit(0);
        } catch (err) {
          writeFailClosedError(err, { phase: 'dispatch', event: input.hook_event_name });
          process.exit(2);
        }
      }
      ```
